# 스토어드의 > 펑션(함수) vs 프로시져
- 스토어드 함수 : 쿼리문 내부에서 쓸 수 있다
- 프로시져 : 쿼리에서 쓸 수 있다
- 함수는 제약조건이 많고
- 프로시져는 제약조건이 적다
- 커스텀은 대부분 프로시져가 편하니까 

```sql
DROP PROCEDURE IF EXISTES SP_HELLO;
DELIMITER $$
CREATER PROCEDURE SP_HELLO()
  BEGIN
    DECLARE STR CHAR(20) DEFAULT 'POPI';
    SET STR = 'HELLO, WORDL!';
    

```
